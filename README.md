# Todo React com Backend Node.js

Um aplicativo simples de lista de tarefas em React com um backend Node.js.

## Tecnologias utilizadas

### Front-end
- Next.js
- Tailwind
- Axios 
- Jwt-decode
- Js-cookie

### Back-end
- Django Framework
- PostgreSQL

Acesse em nosso site: https://notes-amber-tau.vercel.app/
